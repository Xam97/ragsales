# RAGSales - Sistema Inteligente de Análisis de Ventas

RAGSales es un proyecto de inteligencia artificial orientado al análisis de datos de ventas utilizando RAG y Mistral AI.

El objetivo es desarrollar un sistema capaz de comprender información, identificar datos relevantes y responder preguntas de forma inteligente a partir de un dataset de ventas.

---

## Dataset utilizado

Nombre:

`sales_data_sample.csv`

Fuente:

https://www.kaggle.com/datasets/kyanyoga/sample-sales-data

---

## Tecnologías utilizadas

- Python
- Pandas
- NumPy
- LangChain
- Mistral AI
- ChromaDB
- Google Colab
- GitHub

## Avance actual

- Feature engineering (GROSS_VALUE, DISCOUNT_APPLIED, YEAR, MONTH)

- Outliers detectados en cantidades: 8
