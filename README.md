# RAGSales

Sistema inteligente de análisis de ventas capaz de interpretar datos, identificar información relevante y generar respuestas inteligentes mediante RAG y Mistral AI.

## Tecnologías
- Python
- Mistral AI
- RAG
- ChromaDB
- Pandas

## Objetivo
Crear una IA capaz de responder preguntas sobre datos de ventas y comprender información contextual.
